Quality Control (QC)

Assess raw FASTQ files using FastQC and MultiQC. Check for adapters, quality scores, and GC bias. For publication, report QC metrics and thresholds used (e.g., Phred >30).

FastQC Command 
# Run FastQC on raw reads
fastqc -o qc_results/ raw_reads/*.fastq.gz
# Aggregate with MultiQC for summary
multiqc qc_results/ -o multiqc_report/

Best practice: Archive QC reports in supplementary materials for reproducibility.

Metric Threshold for Publication
Per Base Quality >28 across reads
Adapter Content < 5%
GC Content Matches expected for species

Alignment

Align trimmed reads to reference genome using STAR (splice-aware). For human, use GRCh38. Report alignment rates (>80% unique) in methods section.

STAR Alignment 
# Build index if needed
STAR --runThreadN 16 --runMode genomeGenerate --genomeDir genome_index/ --genomeFastaFiles ref.fa --sjdbGTFfile annot.gtf
# Align reads
STAR --runThreadN 16 --genomeDir genome_index/ --readFilesIn trimmed_R1.fastq.gz trimmed_R2.fastq.gz --outFileNamePrefix aligned_ --outSAMtype BAM SortedByCoordinate

For publication: Use --quantMode GeneCounts for direct quantification. Validate with RNA-seq QC tools like RSeQC.

Quantification

Quantify gene/transcript expression using featureCounts or Salmon. Prefer Salmon for isoform-level in complex analyses.

featureCounts Example 
# Count reads per gene
featureCounts -T 16 -a annot.gtf -o counts.txt aligned_*.bam -p -s 2  # For paired-end, stranded
# Or use Salmon for quasi-mapping
salmon quant -i transcriptome_index -l A -1 R1.fastq.gz -2 R2.fastq.gz -o quant_out/ --validateMappings

Publication tip: Report library type (stranded/unstranded) and use tximport to summarize to gene level for DESeq2.

Reading Data

Load count matrix and metadata. Filter low-count genes (e.g., >10 in >2 samples). For sci papers, describe data filtering in methods.

Loading & Filtering 
library(DESeq2)
countData <- read.csv("counts.txt", row.names=1, sep="\t", skip=1)  # Skip featureCounts header
colData <- read.csv("metadata.csv", row.names=1)
colData$condition <- factor(colData$condition)
dds <- DESeqDataSetFromMatrix(countData = countData, colData = colData, design = ~condition + batch)  # Include covariates
keep <- rowSums(counts(dds) >= 10) >= min(table(colData$condition))  # Filter low counts
dds <- dds[keep,]

Variance-stabilized heatmap of top variable genes

Publication: Use vst() for visualization to stabilize variance.

Filter Rationale
Low Counts Reduce noise; report criteria
Outliers Use PCA to detect; justify removal

Differential Expression Genes (DEG) Analysis

Perform DE using DESeq2 with Wald test. Adjust for multiple testing (FDR <0.05). For papers, report model formula and contrasts.

Full DESeq2 Pipeline 
dds <- DESeq(dds)  # Dispersion estimation and testing
res <- results(dds, contrast=c("condition", "treated", "control"), alpha=0.05)  # FDR cutoff
resLFC <- lfcShrink(dds, contrast=c("condition", "treated", "control"), type="apeglm")  # Shrinkage for low counts
degs <- subset(resLFC, padj < 0.05 & abs(log2FoldChange) > 1)
write.csv(as.data.frame(degs), "degs.csv")  # Export for downstream

Volcano plot highlighting significant DEGs

Best practice: Use IHW for independent hypothesis weighting if >10k tests.

Protein-Protein Interaction (PPI) Networks

Construct PPI from DEGs using STRING. Analyze with Cytoscape for modules. Report hub genes and enrichment scores.

STRINGdb & igraph 
library(STRINGdb)
library(igraph)
string_db <- STRINGdb$new(species=9606, score_threshold=700)  # High confidence
mapped <- string_db$map(as.data.frame(rownames(degs)), "SYMBOL", removeUnmappedRows=TRUE)
interactions <- string_db$get_interactions(mapped$STRING_id)
g <- graph_from_data_frame(interactions[, c("from", "to")], directed=FALSE)
clusters <- cluster_louvain(g)  # Module detection

Batch Effect Correction

Detect with PCA; correct using ComBat-seq or limma. Re-run DE post-correction.

sva & limma 
library(sva)
library(limma)
mod <- model.matrix(~condition, data=colData)
mod0 <- model.matrix(~1, data=colData)
svobj <- sva(counts(dds), mod, mod0)  # Estimate surrogates
corrected <- removeBatchEffect(assay(dds), batch=colData$batch, covariates=svobj$sv)
dds_corrected <- dds
assay(dds_corrected) <- corrected

Publication: Show pre/post PCA plots in supplements.
